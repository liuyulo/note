word structure can be represented with morphological trees

        n
    ┌───┴───┐
   adj     suff
 ┌──┴─┐     │
 n   suff   │

job -less -ness

(but this is used here to save space)

   ┌─ adj ┬── n ──── job
n ─┤      └── suff ─ -less

Complex Derivations and Constraints

if both prefix and suffix exist, check which one can freely attach to the n/v/adj/adv/etc

discontinuation - (dis-continua)tion exists because dis- is attached to verb - dis(continua-tion) dne because dis- cannot attach to noun

   ┌─ v ┬── pref ─ dis-
n ─┤    └── v ──── continue
   └─ suff ─────── -tion

Morphological Ambiguity

words that are structurally ambiguous have multiple meanings, therefore multiple trees

  • un: reverse meaning, but the verb must be reversible
    • un(believable) is not ambiguous because unbelieve dne

untie(able): be able to untie

     ┌─ v ┬── pref ─ un-
adj ─┤    └── v ──── tie
     └─ suff ─────── -able

un(tieable): not tieable, unable to tie

     ┌─ pref ──────── un-
adj ─┤      ┌─ v ──── tie
     └─ adj ┴─ suff ─ -able

Word Formation Process

Compounding

join together two existing words into a new unit

  • head determines the lexical category
    • in Eng, ususally on the right (parking ticket, case-sensitive)

Properties of Compounds

written in multitude of ways: hyphen, single word, separate words
pronounce differently: “greenhouse” vs “green house”

Endocentric

meaning is related to the head of the compound

  • earthworm is a type of worm
  • self-care is a type of care

Exocentric

meaning is not predictable

  • bluebell is not a type of bell (it’s a flower)

Other Word Formation Process

Affixation

inflectional derivational
prefixation write
suffixation rewrite writer

Internal Change (Inflection)

a non-morphemic element is changed to represent inflection(grammatical change)

- swim => swam
- drink => drank

Suppletion (Inflection)

replaced by another completely unrelated morpheme

- go => went

Reduplication (Inflection)

Partial Reduplication

repeat part of the morphological base

eg Tagalog

- takbo(run) => tatakbo(will run)
- pili(choose) => pipili(will choose)

Full Reduplication

copy the entire morphological base to change meaning

eg Indonesian

- orang(man) => orang orang(men)
- mangga(mango) => mangga mangga(mangoes)

Zero Derivation

assign a new lexical category to an already existing word

  • talk, sleep, ship, train, stop, drink

Clipping

shorten a multisyllabic word by deleting syllable(s)

- fanatic => fan
- biology => bio
- application => app

Blending

create new words from parts of already existing words

- motor + hotel = motel
- breakfast + lunch = brunch

Backformation

create new lexicas entries by removing an affix from an already existing word

- donation => donate
- editor => edit

Acronym and Initialism

take the first letters in phrase

  • acronym: pronounce as a word
    • NASA: Natioal Aeronautics and Space Administration
    • laser: light amplification by simulated emission of radiation
  • initialism(abbreviation): pronounce letter by letter
    • USA: United States of America
    • CD: Compact Disc

Coinage

produce a new word from scratch, responsible for product names, company names, etc

  • Kleenex, Kodak, Teflon

Eponymy

words created from names of people

- ferries => George Washington Gale Ferries Jr.
- watt => James Watt
- and other measurement units
