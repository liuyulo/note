Causal Inference

causal diagram
  • A→BA\to BA→B means AAA causes BBB

confounder: third variable that causes both

best practices: elaborate theories

if there is causal, it should make several predictions

  • test them, if checks out, then more evidence

Randomised Controlled Experiments

randomised controlled experiments
  • breaks the connection between features of indivitual (i.e. covariate)
  • covariates tend to balance by random chance
    • can check with a balance check
    • also balances unobserved covariates
answer the causal question: does treatment affect outcome
  • estimate average: sensitive to outlier
  • randomised controlled is challenging
    • threats to SUTVA (aka indep)
    • unethical (i.e. do seat belts save lives)
does the new coupon UI makes more money

define measure

  • “total revenue”, because num of user per group can vary
  • revenue/user is better

target population

  • “every one who visits site” valid but noisy
  • “users who completed purchase” wrong, doesnt measure the propotion of purchase
  • “users after start purchase” better
interpret result
  • stat sig: p-values
  • sta power: prob of rejecting null assuming has effect
    • “assume effect, does this exp detects it?”
    • in practice, need 80% power
  • practical sig: depends on context

tips

  • use invariant metrics (shouldnt change between treatment and control)
    • dont change => more confidence
  • check sample ratio
  • design analysis before collect data

avoid

  • dont peek, dont continuously monitor
    • can be insig in end even if sig in middle
  • dont survivorship bias
  • dont SUTVA (e.g. social networks)

Observational Study

natural experiments use existing data to study causal relationships

  • researcher is observer
  • less problem with ethics, cost, feasibility
  • more problem with validity, sampling
    • e.g. treatment assignment can be caused by unobserved covariate (aka confounders)

Matching Studies

matching studies pick pairs of similar people, 1 treated and 1 control

  • e.g. one lived, one died in same car

    • controlled car, speed, serverity of accident, etc

  • only combine matches with same prob of treatment

    • otherwise result changes
problems with matching
  • match might not be possible
    • solution: propensity scores (a single number)
    • well defined regardless how true model is
    • if model true, then matching propensity as good as match with features
    • (usually logistic regression on features)
  • model is rarely true
    • sensitivity analysis
    • measures “how bad would the naive model have to be in order to change my conclusions”
