W3 Data Summerisation

descriptive statistics
  • arithmetic mean: single number closest to all values
  • variance: how spread out values are
  • std deviation: amount of variation around mean (same unit as mean)

robust statistic (single data change won’t affect the stat)

  • median
  • quartile
  • interquartile range: between 25% and 75%

heavy-tailed distribution (not nomal)

  • follows power law, that is x−αx^{-\alpha}x−α
  • unrobust stats (mean, var) are useless

use statistic tests to check if model fits with distribution

  • normality test: test if normal distribution
  • KS-test: test if empirical data fits with model
quantifying uncertainty

datasets are sample from a population

  • has variance: variation among samples
  • has bias: systematic deviation from population

inductive reasoning

  • data -> sample: measurement
    • reliable: consistent and repeatable
    • valid: no systematic bias
  • sample -> study population: internal validity
    • does the sample accurately reflect the population
    • use random sampling
  • study population -> target population: external validity

null hypothesis: if nothing interesting happens

  • p-value: probability of obtaining this result under the assumption that null is correct
  • low p -> small data -> big effect
    • so any big data can have small p, even with small effect
  • statistically significant: likely to be detectable, but not necessarly large

confidence interval (γ\gammaγ CI): [m−γ,m+γ][m-\gamma,m+\gamma][m−γ,m+γ] where mmm is empirical estimate

  • range of estimates reasonable by the observed data
  • reject the null “the true value is μ0\mu_0μ0​” for any μ0∈[m−γ,m+γ]\mu_0\in[m-\gamma,m+\gamma]μ0​∈[m−γ,m+γ]

W4 Data Visualisation

visualisation

visually encodes complex information, to be decoded by audience

motivation

  • analysis: find relationship/structure
  • communication: tell story visually

criteria

  • expressiveness: express all and only the facts in the data
  • effectiveness: easily decodable

feature

  • dimension/feature: independent variable
  • measure/outcome: dependent variable
  • assign data fields (nominal/ordinal/quantitative) to data channels (x/y/color) to produce graphical marks (points/bars/lines)
visualisations by data types

nominal (categorical, labels)

  • line chart: connecting points suggests there are stuff between points
  • color/size: hard to decode

ordinal (has order)

quantitative (interval, ratio)

  • histogram: detects skew easily
  • box plot: has quartiles, outliers
    • but different distrs can have same box plot
  • violin plot: box plot but shows distribution
multivariate

  • scatter plot: shows relationship between 2 vars

  • line plot, stacked plot

  • more than two: dimentionality reduction

    • PCA: perserve global structure
    • t-SNE: preserve local structure (nearest neighbors)
principles
  • use consistent axes
  • use axes labels
  • choose axis scale
    • option 1: clilp outliers and mention them
    • option 2: log scale
      • for long tails, emphasize change, not linear change
  • color scale
    • sequential: ordinal/quantitative
      • increasing brightness
    • diverging: ratio
      • data has natural midpoint
      • neutral color for midpoint, saturated for endpoint
    • categorical: nominal
      • lots of contrast between each adjacent color
