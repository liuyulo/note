12 Illness

Grammar.
  • 〜んです　src
    • 説明・理由を求める/述べる
    • 動詞/形容んです
    • 名詞/形動なんです
  • 〜すぎる too much
    • 動詞：　連用音便なし＋すぎる
    • 形容/形動：　語幹＋すぎる
  • 動詞：　ほうがいいです
    • 連用:音便あり＋たほうがいいです better to do
    • 未然＋ないほうがいいです better not to do
  • 〜ので
    • same as から
    • don’t use keigo
    • 名詞/形動なので
  • しなければいけません must do
    • しなくちゃいけません
    • しなくてはいけません
    • しないといけません
    • しなきゃいけません
  • {動/名/形容/形動}でしょうか would it?

Grammar.
  • given a domain D with comparable property P and A,B,C ∈ D
    • compare 2 elements
      • BよりAの方がPです: P(A) > P(B)
      • AとBと、どっちの方がPですか: Between A and B, which is more P?
    • compare more
      • AとBとCの中でAが一番Pです: P(A) >= P(B),P(C)
      • Dの中でAが一番Pです: ∀B ∈ D P(A) >= P(B)
      • Qが一番Pですか where Q is the question word suitable for items in D
  • {...}の as a new noun to avoid repetition
    • translates to “the one that is {...}”
  • {V present}つもりだ intend to do
  • {QW}か{P} vs {QW}{P}も
    • replaces はがを, same rule as も
  • 〜で as tool (translates using propositions)
  • かかる has no particle
