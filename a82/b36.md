\end{align*}\quad\begin{align*}

## 6.9 Prenex Normal Form
<details class="definition" open><summary>`F` as a FO formula is in **PNF** iff</summary>
$$F=Q_1x_1\dots Q_kx_x E$$

for some
: $Q_i$ as quantifiers
: $E$ as a FO formula without quantifiers

</details>
<details class="theorem" open><summary>PNF Theorem</summary>
Any predicate formula has a PNF equivalent
</details>

## 6.5 Logical Impl and Logical Equiv
<details class="definition" open><summary>Analogy to Propositional Formulas</summary>
for `F` as a predicate

- `F` is **valid** `<=>` all interpretations satisfies `F`
- `F` is **satisfiable** `<=>` some interpretations satisfies `F`
- `F` is **falsifiable** `<=>` some falsifies `F`
- `F` is **unsatisfiable** `<=>` all falsifies `F`
</details>
<details class="definition" open><summary>LIMP and LEQV of Predicates</summary>
for any `A` and `B` as FO formulas

- `A ==> B` iff all interp that satisfies `A` also satisfies `B`
- `A <=> B` iff `A ==> B` and `B ==> A`
</details>
<details class="theorem" open><summary>Logical Equivalences (same applies for `∃` with `∀`)</summary>
don't make free variables quantified
```
   ¬∀x A <=> ∃x ¬A
 ∀xA ∧ B <=> ∀x(A ∧ B)
 ∀xA ∨ B <=> ∀x(A ∨ B)
∀xA -> B <=> ∃x(A -> B)           (caution!)
A -> ∀xB <=> ∀x(A -> B)
```

can rename variables 
</details>

## 6.2 First-Order Language Syntax
<details class="definition" open><summary></summary>
FO language consists of

- `{ x, y, ... }` infinite set of **variables** 
- `{ A, B, ... }` finite set of **predicate symbols** (each with arity)
- `{ a, b, ... }` finite set of **constant symbols** 
-  `{ ¬, ∧, ∨, -->, <-> }` **connectives**
- `{ ∀, ∃ }` **quantifiers** 
- `{ (, ) }` **parentheses** 

set $\mathcal L$ of FO formulas is the smallest set st (with unique parse tree)

- base: **atomic formulas** of $\mathcal L\in\mathcal L$
- IS: if $F_1,F_2\in\mathcal L$ and $x$ is a variable then 
  - $\neg F_1,(F_1\land F_2),(F_1\lor F_2),(F_1\to F_2),(F_1\lrarr F_2)\in\mathcal L$
  - $\forall xF_1,\exists xF_i\in\mathcal L$
</details>
<details class="definition" open><summary>Free Variable wrt FO formula `F`</summary>
with respect to `F`, an occurence of var `x` is **free** iff `x` is not quantified in `F` towards the highest level of the parse tree

- **sentence**: predicate without any free variables

# 5 Propositional Logic
> `PV` is set of propositional variables<br>
> `{ ¬, ∨, ∧, -->, <-> }` is set of connectives

## 5.2 Proposition
<details class="definition" open><summary>Propositional Formulas $\mathcal F_{PV}\in(PV\cup\{\neg,\land,\lor,\rarr,\lrarr\})^\ast$</summary>
given $PV$, define $\mathcal F_{PV}$ as the smallest set such that

- base: $x\in PV\implies x\in\mathcal F$
- IS: $P_1,P_2\in\mathcal F_{PV}\implies$
  - $\neg P_1,(P_1\land P_2),(P_1\lor P_2)\in\mathcal F_{PV}$
  - $(P_1\rarr P_2),(P_1\lrarr P_2)\in\mathcal F_{PV}$

</details>
<details class="definition" open><summary>Truth Assignment to $PV$ is $\tau:PV\to\{0,1\}$</summary>
given $\tau:PV\to\{0,1\}$, define extended T.A. $\tau^\ast:\mathcal F_{PV}\to\{0,1\}$ 

- base: $P\in PV\implies\tau^\ast(P)=\tau(P)$
- IS: let any $P\not\in PV$
  - assume $\exist Q_1,Q_2\in\mathcal F_{PV}$ such that $\tau^\ast(Q_1)$ and $\tau^\ast(Q_2)$ are defined
  - $\tau^\ast(\neg Q_1)=1-\tau^\ast(Q_1)$
  - $\tau^\ast((Q_1\land Q_2))=\tau^\ast(Q_1)\tau^\ast(Q_2)$
  - $\tau^\ast((Q_1\lor Q_2))=\tau^\ast(Q_1)+\tau^\ast(Q_2)-\tau^\ast(Q_1)\tau^\ast(Q_2)$
  - $\tau^\ast((Q_1\rarr Q_2))=1-\tau^\ast(Q_1)+\tau^\ast(Q_1)\tau^\ast(Q_2)$
  - $\tau^\ast((Q_1\lrarr Q_2))=1-\tau^\ast(Q_1)+\tau^\ast(Q_2)$
</details>

<details class="theorem" open><summary>5.3 Truth Table</summary>
Used to determine $\tau^\ast$ for every possible T.A. $\tau$ of an $P\in\mathcal F$
</details>

## 5.4 Satisfiable vs Falsifiable
<details class="definition" open><summary>Given $P\in\mathcal F$</summary>
- $\small\tau$ **satisfies** $\small P\iff\tau^\ast(P)=1\iff\tau^\ast(P)\ne0\iff$ not **falsifies** $\small P$
- $P$ is **satisfiable** $\iff\exists\tau\enspace\tau$ satisfies $P$
- $P$ is **tautology** $\iff\forall\tau\enspace\tau$ satisfies $P\iff P$ is NOT **falsifiable**
- $P$ is **unsatisfiable** $\iff\nexists\tau\enspace\tau$ satisfies $P\iff P$ is NOT **satisfiable**
</details>

<details class="definition" open><summary>Logical Implication and Logical Equivalence</summary>

for $P,Q\in\mathcal F$
: `(P ==> Q)` iff `P -> Q` is tautology
: `(P <=> Q)` iff `P ==> Q` and `Q ==> P`

- `P --> Q` is a propositional formula
- `P ==> Q` is a proposition
</details>
<details class="theorem" open><summary>Logical Equivalences</summary>
- Double Negation
- De Morgan's x2
- Communatative x2
- Associative x2
- Distributive x2
- Identity x2
- Indempotency x2
- `-->` and `<->` Laws
</details>

## 5.9 Normal Forms
<details class="definition" open><summary>Restricted Types of Propositional Formulas</summary>
- **literal** `==>` contains `PV` or `¬`
- **term** `==>` contains conjuctions of literals
- **clause** `==>` contains disjuctions of literals
- **CNF** `<=>` conjuction of clauses (i.e. a conj of disjs)
- **DNF** `<=>` disjuction of terms (i.e. a disj of conjs)
</details>
<details class="theorem" open><summary>DNF/CNF Theorem</summary>
$\forall P\in\mathcal F\enspace\exists$ DNF $P_D$ and CNF $P_C$ st $P\iff P_D\iff P_C$
</details>
<details class="algorithm" open><summary>Make DNF from Truth Table of $P\in\mathcal F$</summary>
- make truth table
- for each $\tau_i$ st $\tau_i^\ast(P)=1$, make **term** $F_i$ st $\tau_i$ **satisfies** $F_i$
- make DNF as the disjuction of $F_i$
</details>
<details class="algorithm" open><summary>Make CNF from Truth Table of $P\in\mathcal F$</summary>
- make truth table
- for each $\tau_i$ st $\tau_i^\ast(P)=0$, make **clause** $F_i$ st $\tau_i$ **falsifies** $F_i$
- make CNF as the conjuction of $F_i$
</details>

## 5.11 Completeness
<details class="definition" open><summary>Boolean function $f:\{0,1\}^n\to\{0,1\}$</summary>
$P\in\mathcal F$ **represents** $f\iff\forall\tau\enspace\tau^\ast(F)=f(\tau(x_i))$
: all boolean functions can be represented (from truth table)

</details>
<details class="definition" open><summary>Completeness of a set of **connectives**</summary>
a set of connectives is **complete** iff any bool fn can be represented by $P\in\mathcal F$ using only connectives (uoc) from the set
</details>
<details class="theorem" open><summary>Complete Connectives</summary>
- `{ ¬, ∨ }` and `{ ¬, ∧ }` are complete
</details>
<details class="algorithm" open><summary>Prove `C` as set of connectives is Complete</summary>

1. (choose `{ ¬, ∨ }` xor `{ ¬, ∧ }`)
1. (find a LEQV of `¬P` and a LEQV of `P ∧ Q` that uoc C)
2. define $\mathcal G_{PV}$ as the smallest set such that
    - base: `x ∈ PV ==> x ∈ G`
    - IS: `P, Q ∈ G ==> ¬P, (P ∧ Q) ∈ G`
3. structural induction on def of `∀F uoc G, ∃F' uoc C st F <=> F'`
    - base: `F = x ∈ PV` let `F' = x uoc C`
    - IS: let any `P,Q ∈ G - PV`
      - assume `∃P',Q' uoc C` and `P <=> P'` and `Q <=> Q'` as IH
      - case `F = ¬P`, let `F' = {something with P' uoc C}`
        - `∴ F' uoc C [IH]`
        - `∴ {F' <=> F} [IH]` as wanted
      - case `F = (P ∧ Q)`, let `F' = {sth with P' and Q' uoc C}`
        - `∴ F' uoc C [IH]`
        - `∴ {F' <=> F} [IH]` as wanted
4. `∵ { ¬, ∧ }` is complete<br>`∴ C` is complete
</details>
<details class="algorithm" open><summary>Prove `C` as a set of connectives is NOT complete</summary>
1. (choose `{ ¬, ∨ }` xor `{ ¬, ∧ }`)
2. (find `P(F)` that is true `∀F` uoc C, but falsifiable by `F uoc { ¬, ∧ }`)
3. define $\mathcal G_{PV}$ as the smallest set that uoc C
4. define `P(F)`
5. structural induction on def of `F uoc C` to show `∀F ∈ G, P(F)`
6. find `F uoc { ¬, ∧ }` that falsifies `P(F)`
7. `∴ C` is not complete
</details>
<details class="example" open><summary>`{ ↓ } = { NOR }` is complete</summary>
- `¬P <=> P ↓ P`
- `P ∨ Q <=> ¬(P ↓ Q) <=> (P ↓ Q) ↓ (P ↓ Q)`
</details>

## 7.6 The Pumping Lemma
<details class="theorem" open><summary>The Pumping Lemma</summary>
if $L\sube\Sigma^\ast$ is a regular lang, then $\exist n>0$ st $\forall x\in L\enspace\exist u,v,w\in\Sigma^\ast$ st all
: $x=uvw$
: $v\ne\varepsilon$
: $|uv|\le n$
: $\forall k\in\mathbb N\enspace uv^kw\in L$

</details>
<details class="example"><summary>choose good string: $\small L=\{x:\texttt{x.count(1) == x.count(0)}\}$</summary>
by way of contradiction, assume $L$ is regular
: $\therefore\exists n\in\mathbb N$ as in the Pumping Lemma

let $x=0^n1^n$
: $\therefore x\in L$ and $|x|=2n\ge n$ by def of $L$

$\therefore$ by Pumping Lemma, $\exists u,v,w\in\Sigma^\ast$ st
: (1) $x=uvw$
: (2) $v\ne\varepsilon$
: (3) $|uv|\le n$
: (4) $\forall k\in\mathbb N\enspace uv^kw\in L$

$$\begin{align*}
  \because&\, uv=0^m\quad 0\le m\le n\tag{\sf by 1 3}\\
  \therefore&\, v=0^k\quad 0<k\le n\tag{\sf by 2}\\
  \therefore&\,uv^2w=uvvw=0^{n+k}1^n\\
  \therefore&\, uv^2w\not\in L\tag{$k>0\implies n+k\ne n$}\\
  \therefore&\,\textsf{contradicts (4)}\\
  \therefore&\, L\textsf{ is not regular}
\end{align*}$$
</details>
<details class="example"><summary>by cases and choose good $k$: $L=\{10^k1:k\textsf{ is prime}\}$</summary>
by the way of contradiction, assume $L$ is regular<br>
$\therefore\exists n>0$ as in Pumping Lemma<br>
let $x=10^p1$ where $p>n$ is a prime<br>
$\therefore x\in L$ by def and $|x|=p+2>p>n$<br>

$\therefore\exists u,v,w\in\Sigma^\ast$ st
: 1. $x=uvw$
: 2. $v\ne\varepsilon$
: 3. $|uv|\le n$
: 4. $\forall k\in\mathbb N\quad uv^kw\in L$

- case $u=\varepsilon$: 
$$\begin{align*}
  &\therefore\exists i<n\textsf{ st }v=10^i\quad w=0^{p-i}1\\
  &\because uv^0w\in L\tag{\sf by 4}\\
  \textsf{but }&uv^0w=w=0^{p-i}1\not\in L\tag{$\nexists x\in L$ st $\#_1(x)=1$}\\
  &\therefore\textsf{contradicts }L\textsf{ is regular}
\end{align*}$$

- case $u\ne\varepsilon$
$$\begin{align*}
  &\therefore\exists0<j<i<n\textsf{ st }u=10^{i-j}\quad v=0^j\quad w=0^{p-i}1\\
  &\because uv^{j(p+1)}w\in L\tag{\sf by 4}\\
  &\textsf{but }uv^{j(p+1)}w=1^{i-j+j(p+1)+p-i}=10^{p(j+1)}1=10^k1\\
  &\textsf{where }k\textsf{ is not prime}\\
  &\therefore uv^{j(p+1)}w\not\in L\\
  &\therefore\textsf{contradicts }L\textsf{ is regular}
\end{align*}$$

</details>

## 7.5 Closure under $\mathcal L$
<details class="algorithm" open><summary>Prove $f$ is closed under $\mathcal P(\Sigma^\ast)$ (Regex)</summary>
define for $R\in\mathcal{RE}$
: $P(R):\exist R'$ st $f(\mathcal L(R))=\mathcal L(R')$

base
: $P(\varnothing),P(\varepsilon),P(a)\enspace\forall a\in\Sigma$

IS let any $S,T\in\mathcal{RE}$
: assume $\exist S',T'$ st $f(\mathcal L(S))=\mathcal L(S')\enspace f(\mathcal L(T))=\mathcal L(T')$
: wts $P(S+T)\enspace P(ST)\enspace P(S^\ast)$

</details>
<details class="algorithm" open><summary>Prove $f$ is closed under $\mathcal P(\Sigma^\ast)$ (FSA)</summary>

define for DFSA $M$
: $P(M):\exist$ FSA $M'$ st $f(\mathcal L(M))=\mathcal L(M')$

</details>
<details class="theorem" open><summary>Known Closures</summary>
if $L$ and $L'$ are regular, then these are also regular

|
|:-:|:-:|:-:|:-:|:-:|
| $\overline L$ | $L\cup L'$ | $L\cap L'$ | $LL'$ | $L^\ast$

</details>

<details class="example"><summary>$f(L)=\overline L=\Sigma^\ast-L$</summary>
WLOG let any DFSA $M=(Q,\Sigma,\delta,s,F)$
: choose DFSA $\overline M=(Q,\Sigma,\delta,s,Q-F)$

$$\begin{align*}
  \therefore x\in\mathcal L(\overline M)&\iff\delta^\ast(s,x)\in Q-F\tag{\sf def of AC}\\
  &\iff\delta^\ast(s,x)\not\in F\tag{\sf def of $\overline M$}\\
  &\iff x\not\in\mathcal L(M)\tag{\sf def of AC}\\
  \therefore \mathcal L(\overline M)&=\overline{\mathcal L(M)}=\overline L\textsf{ as wanted}
\end{align*}$$
</details>


<details class="example"><summary>$f(L)=\{u0v:u,v\in\Sigma^\ast,uv\in L\}$</summary>
let $\Sigma=\{0,1\}$
: define $f(L)=\{u0v:u,v\in\Sigma^\ast,uv\in L\}$
: wts $P(R):\exist R'$ st $\mathcal L(R')=f(\mathcal L(R))$

**base**
: case $R=\varnothing$: let $R'=\varnothing$
$$\begin{align*}
  \therefore\mathcal L(R)&=\varnothing\\
  \therefore f(\mathcal L(R))&=\varnothing\tag{\sf def of $f$}\\
  &=\mathcal L(R')\textsf{ as wanted}
\end{align*}$$

: case $R=\varepsilon$: let $R'=0$
$$\begin{align*}
  \therefore\mathcal L(R)&=\{\varepsilon\}\\
  \therefore f(\mathcal L(R))&=\{0\}\tag{\sf def of $f$}\\
  &=\mathcal L(R')\textsf{ as wanted}
\end{align*}$$

: case $R=b\enspace\forall b\in\Sigma$: let $R'=0b+b0$
$$\begin{align*}
  \therefore\mathcal L(R)&=\{b\}\\
  \therefore f(\mathcal L(R))&=\{0b,b0\}\tag{\sf def of $f$}\\
  &=\mathcal L(R')\textsf{ as wanted}
\end{align*}$$

**IS**: let any $S,T\in\mathcal{RE}$
: assume $\exist S',T'$ st $\mathcal L(S')=f(\mathcal L(S))$ and $\mathcal L(T')=f(\mathcal L(T))$ as IH
: case $R=S+T$: let $R'=S'+T'$
$$\begin{align*}
  \therefore f(\mathcal L(R))&=f(\mathcal L(S+T))\\
  &=f(\mathcal L(S)\cup\mathcal L(T))\tag{\sf prop of $\mathcal L$}\\
  &=f(\mathcal L(S))\cup f(\mathcal L(T))\tag{\sf maturity}\\
  &=\mathcal L(S')\cup\mathcal L(T')\tag{\sf IH}\\
  &=\mathcal L(S'+T')=\mathcal L(R')\textsf{ as wanted}
\end{align*}$$
: case $R=ST$: let $R'=S'T+ST'$
$$\begin{align*}
  \therefore f(\mathcal L(R))&=f(\mathcal L(ST))\\
  &=f(\mathcal L(S)\mathcal L(T))\tag{\sf prop of $\mathcal L$}\\
  &=f(\mathcal L(S))\mathcal L(T)\cup\mathcal L(S)f(\mathcal L(T))\tag{\sf maturity}\\
  &=\mathcal L(S')\mathcal L(T)\cup\mathcal L(S)\mathcal L(T')\tag{\sf IH}\\
  &=\mathcal L(S'T+ST')=\mathcal L(R')\textsf{ as wanted}\\
\end{align*}$$
: case $R=S^\ast$: let $R'=0+S^\ast S'S^\ast$

$$\begin{align*}
  \therefore f(\mathcal L(R))&=f(\mathcal L(S^\ast))=f(\mathcal L(S)^\ast)\tag{\sf prop of $\mathcal L$}\\
  &=f(\{\varepsilon\})\cup\mathcal L(S)^\ast f(\mathcal L(S))\mathcal L(S)^\ast\tag{\sf maturity}\\
  &=\{0\}\cup\mathcal L(S^\ast)\mathcal L(S')\mathcal L(S^\ast)\tag{\sf IH}\\
  &=\mathcal L(0)\cup\mathcal L(S^\ast S'S^\ast)\tag{$\mathcal L(0)=\{0\}$}\\
  &=\mathcal L(0+S^\ast S'S^\ast)=\mathcal L(R')\textsf{ as wanted}
\end{align*}$$

$\therefore$ by structural induction $P(R)$ holds for any regex $R\quad\blacksquare$
</details>

## 7.4 NFSA
<details class="definition" open><summary>NFSA $M=(Q,\Sigma,\delta,s,F)$</summary>
where
: $Q$ finite set of states
: $\Sigma$ finite input alphabet (set of symbols)
: $\delta:Q\times(\Sigma\cup\{\varepsilon\})\to\mathcal P(Q)$ transition function $\delta(q,a)=q'$

tl;dr its DFSA but

## 7.3 DFSA
<details class="definition" open><summary>DFSA $M=(Q,\Sigma,\delta,s,F)$</summary>
where
: $Q$ finite set of states
: $\Sigma$ finite input alphabet (set of symbols)
: $\delta:Q\times\Sigma\to Q$ transition function $\delta(q,a)=q'$
: $s\in Q$ initial/start state
: $F\sube Q$ accepting states

</details>
<details class="definition" open><summary>Extended transition function $\delta^\ast:Q\times\Sigma^\ast\to Q$</summary>

  \delta(\delta^\ast(q,y),a)&x=ya\quad y\in\Sigma^\ast\; a\in\Sigma\\

</details>

<details class="example"><summary>Structural induction: $(xy)^R=(y)^R(x)^R$</summary>

definition of reverse

<br><br>

  &=(x)^R\tag{$\forall w\enspace w\varepsilon=w$}\\
  &=\varepsilon(x)^R\tag{$\forall w\enspace \varepsilon w=w$}\\
  &=(y)^R(x)^R\textsf{ as wanted}\tag{$(y)^R=\varepsilon$}\\
\end{align*}

  - assume $P(y')$ holds as IH
  - wts $\forall a\in\Sigma\enspace y=y'a\implies P(y)$

    &=(y)^R(x)^R\textsf{ as wanted}\tag{$y=y'a$}\\

	&=(\mathrm{op}(e_1)+\mathrm{op}(e_2))+2\\
	&=(\mathrm{op}(e)-1)+2\tag{\textsf{by (1)}}\\

  \end{align*}

  &=2(2f(n/4)+n)+2n\\

## Binary Search
<details class="definition" open><summary>unimodal slice</summary>
given

- `let v: Vec<i32>`{.rs}
- `let (a, b): (usize, usize)`{.rs} st `0 <= a < b <= v.len()`{.rs}

`v[a..b]`{.rs} is **unimodal** iff `∃ m: usize`{.rs} st

- `a <= m < b`{.rs}
- `v[a..m + 1]`{.rs} increases
- `v[m..q]`{.rs} decreases

then `mode(v[p..q]) == m`{.rs}

- `v` is unimodal `<=>` `v[0..v.len()]`{.rs} is unimodal
- `v[a..b]`{.rs} increases `==>` `m == b - 1`{.rs}
- `v[a..b]`{.rs} decreases `==>` `m == a`{.rs}
- `mode(v[a..a + 1]) == a`{.rs}
- any subslice of unimodal slice is unimodal
- `v.max() == v[m]`{.rs}
</details>

<details class="theorem" open><summary>Python Lemma 2.2</summary>
$$\forall a,b\in\mathbb Z\enspace a+1<b\implies a<\left\lfloor\frac{a+b}2\right\rfloor<b$$
</details>

<details class="example"><summary>Prove iter correct</summary>

```{.rs .numberLines}
fn max(s: &[i32], st: usize, ed:usize) -> i32 {
    // precond: 0 <= st < ed <= s.len()
    //          s[st..ed] is unimodal
    // postcond: return s.max()
    let (mut a, mut b) = (st, ed);
    while a + 1 < b{
        let m: usize = (a + b) / 2;
        if s[m - 1] < s[m] {
            a = mid;
        } else {
            b = mid;
        }
    }
    s[a]
}
```

**LI**

1. `st <= a < b <= ed`{.py}
    - $\therefore$ `s[a:b]`{.py} is unimodal
2. `mode(s[a:b]) == mode(s[st:ed])`{.py}

<details><summary>[Prove LI]{.y}</summary>
- base: on loop entry
\begin{align*}
\because&\,a=\mathrm{st}\enspace b=\mathrm{ed}\tag{\sf line 5}\\
\therefore&\,s[a:b]=s[\mathrm{st}:\mathrm{ed}]\\
\therefore&\,\textrm{mode }s[a:b]=\textrm{mode }s[\textrm{st}:\textrm{ed}]\\
\therefore&\,\textsf{LI2 holds}\\
\\
\because&\,\textrm{st}<\textrm{ed}\tag{\sf precond}\\
\therefore&\,\textrm{st}\le a<b\le\textrm{ed}\\
\therefore&\,\textsf{LI1 holds}
\end{align*}

- IS: for any iter
  - assume LI holds before iter as IH
  - wts LI holds after iter
    - case `s[m - 1] < s[m]`{.py}
  \begin{align*}
    \because a'=m&=\left\lfloor\frac{a+b}2\right\rfloor\quad b'=b\tag{\sf line 3,4}\\
    \therefore\textrm{mode }&s[\textrm{st}:\textrm{ed}]=\textrm{mode }s[a:b]\tag{\sf IH LI2}\\
    &=\textrm{mode }s[m:b]\tag{\sf unimodal}\\
    &=\textrm{mode }s[a':b']\\
    \therefore&\,\textsf{LI2 holds as wanted}\\
    \\
    \because a&<m=a'<b'=b\tag{\sf PL}\\
    \because\textrm{st}&\le a<b<\textrm{ed}\tag{\sf IH LI1}\\
    \therefore\textrm{st}&\le a<a'<b'=b<\textrm{ed}\\
    \therefore&\,\textsf{LI1 holds as wanted}
  \end{align*}
    - case `s[m - 1] > s[m]`{.py}: same structure
    - case `s[m - 1] == s[m]`{.py}: impossible because `s` is unimodal
</details>

<details><summary>[Partial Correctness]{.y}</summary>
- assume termination and LI
- wts postcond
\begin{align*}
  \because a+1&\le b\tag{\sf LI1}\\
  a+1\ge b\tag{\sf line 6}\\
  \therefore a+1&=b\\
  \therefore \textrm{mode }s[\textrm{st}:\textrm{ed}]&=\textrm{mode }s[a:b]\tag{\sf LI2}\\
  &=\textrm{mode }s[a:a+1]\\
  &=a\tag{\sf prop of modal}\\
  \therefore\textrm{max}(s)&=s[a]\tag{\sf prop of modal}\\
  &\textsf{is returned as wanted}\tag{\sf line 14}
\end{align*}
</details>

<details><summary>[Termination]{.y}</summary>
let $e=b-a$

- wts $e\in\mathbb N$
  \begin{align*}
    \because a&<b\tag{\sf LI1}\\
    \therefore e&=b-a>0\\
    \therefore e&\in\mathbb N\textsf{ as wanted}
  \end{align*}
- wts $e'<e$
  - for any iter
    - case `s[m - 1] < s[m]` (line 8)
  \begin{align*}
    e'&=b'-a'\tag{\sf def of $e$}\\
    &=b-m\tag{\sf by (Prove LI)}\\
    &<b-a\tag{\sf PL}\\
    &=e\tag{\sf def of $e$}\\
  \end{align*}
    - (similar to other cases)
- $\therefore$ program terminates
</details>
</details>

# 1 Induction
> $0\in\N$

<details class="definition" open><summary>Principle of Simple Induction (**PSI**)</summary>
to prove $\forall n\geqslant b\enspace P(n)$

- base $P(b)$
- induction step $\forall n\geqslant b\enspace P(n)\implies P(n+1)$
</details>
<details class="example"><summary>show $\forall n\geqslant18\enspace\exist a,b\in\N$ st $4a+7b=n$</summary>
let $P(n):\exist a,b\in\N\textsf{ st }4a+7b=n$<br>

base: $P(18)$
: let $a=1\enspace b=2$
: $\therefore 4a+7b=4\cdot1+7\cdot2=18$
: $\therefore P(18)$ holds

IS: let $n\geqslant18$
: assume $P(n)$ i.e. $\exist a,b\in\N$ st $4a+7b=n$ (IH)
: wts $P(n+1)$ i.e. $\exist c,d\in\N$ st $4c+7d=n+1$

- case $b>0$
\begin{align*}
  \textsf{let }&c=a+2\enspace d=b-1\in\N\\
  \therefore&\,4c+7d=4(a+2)+7(b-1)=4a+8+7b-7\\
  &=4a+7b+1=n+1\textsf{ as wanted}\tag{\textsf{IH}}\\
\end{align*}
- case $b=0$
\begin{align*}
  \therefore&\,4a=n\geqslant18\tag{\textsf{IH}}\\
  \therefore&\,a\geqslant\lceil18/4\rceil=5\tag{$a\in\N$}\\
  \textsf{let }&c=a-5\in\N\enspace d=b+3\\
  \therefore&\, 4c+7d=4(a-5)+7(b+3)=4a-20+7b+21\\
  &=4a+7b+1=n+1\textsf{ as wanted}\tag{\textsf{IH}}
\end{align*}
- $\therefore\forall n\geqslant18\enspace P(n)$ by PSI $\quad\blacksquare$

</details>
<details class="definition" open><summary>Principle of Complete Induction (**PCI**)</summary>
to prove $\forall n\geqslant b\enspace P(n)$

- base $P(b)\enspace P(b+1)\cdots$ (prove $k$ base cases)

<details class="example"><summary>show $\forall n\geqslant18\enspace\exist a,b\in\N$ st $4a+7b=n$</summary>
let $P(n):\exist a,b\in\N$ st $4a+7b=n$<br>

base (show $k=4$ base cases)
: show $P(18)\enspace P(19)\enspace P(20)\enspace P(21)$ holds

IS let any $n\geqslant 18+4=22$
: assume $\forall 18\leqslant i<n\enspace P(i)$ holds
: wts $P(n)$ holds i.e. $\exist c,d\in\N$ st $4c+7d=n$

$$\begin{align*}
  \because&\,n\geqslant22\\
  \therefore&\,18\leqslant n-4\leqslant n\\
  \therefore&\,P(n-4)\textsf{ holds}\tag{\textsf{IH}}\\
  \therefore&\,\exist a,b\in\N\textsf{ st }4a+7b=n-4\\
  \textsf{let }&c=a+1\enspace d=b\\
  \therefore&\,4c+7d=4(a+1)+7b\\
  &=4a+7b+4=n-4+4=n\textsf{ as wanted}\tag{\textsf{IH}}
\end{align*}$$

$\therefore\forall n\geqslant18\enspace P(n)$ by PCI $\quad\blacksquare$
