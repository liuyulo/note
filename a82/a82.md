<details class="leek" open><summary>**Grammar.**</summary>
- given a domain `D` with comparable property `P` and `A,B,C ∈ D`
  - compare 2 elements
    - `B`より`A`の方が`P`です: `P(A) > P(B)`
    - `A`と`B`と、**どっちの方**が`P`ですか: Between `A` and `B`, which is more `P`?
  - compare more
    - `A`と`B`と`C`の中で`A`が一番`P`です: `P(A) >= P(B),P(C)` 
    - `D`の中で`A`が一番`P`です: `∀B ∈ D P(A) >= P(B)` 
    - **Q**が一番`P`ですか where `Q` is the question word suitable for items in `D`
- `{...}の` as a new noun to avoid repetition
  - translates to "the one that is `{...}`"
- `{V present}`つもりだ intend to do
- {QW}か{P} vs {QW}{P}も
  - replaces はがを, same rule as も
- 〜で as tool (translates using propositions)
- かかる has no particle
</details>
