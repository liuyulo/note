# Remainder
- $\textcolor{aqua}{n|a}$ means `a % n == 0`
- $\textcolor{aqua}{a\equiv_nb}$ means `a % n == b % n`
- $\textcolor{aqua}{a\equiv r\mod b}$ means `a % b == r`

# Prime
- $n$ is not a prime iff $\exist x,y\in\Z$ st $n=xy$
- all integers greater than 1 must be a prime or a composite number
